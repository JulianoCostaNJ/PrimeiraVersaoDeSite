# 📚 Aulas de HTML, CSS e JavaScript

Este repositório foi criado para organizar e armazenar materiais de estudo relacionados ao desenvolvimento web utilizando **HTML**, **CSS** e **JavaScript**.

O objetivo é registrar a evolução das aulas, exemplos práticos e exercícios desenvolvidos durante o aprendizado.

---

## 📂 Organização do Repositório

As aulas estão **organizadas em pastas**, facilitando a navegação e o acompanhamento do conteúdo estudado.

Cada pasta representa uma **aula ou conjunto de exercícios**, contendo os arquivos necessários como:

- `index.html`
- `style.css`
- `script.js`
- arquivos auxiliares

Essa estrutura permite acompanhar a progressão das aulas de forma clara.
📦 repositorio
┣ 📂 aula01
┣ 📂 aula02
┣ 📂 aula03
┣ 📂 aula04
┗ 📂 ...

## 🚀 Tecnologias Utilizadas

Durante as aulas são utilizadas as seguintes tecnologias:

- **HTML5** — Estrutura das páginas
- **CSS3** — Estilização e layout
- **JavaScript** — Interatividade e comportamento da página
